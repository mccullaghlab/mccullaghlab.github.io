2.3.1.1. For loops in python#

Loops are a common structure in coding/scripting.

for i in range(10):
    print(i)

Loops are often used to compute sums. For example if we want to know \(\sum_{i=0}^{10} i\)

partialSum = 0
for i in range(11):
    partialSum += i
    print(i, partialSum)
print("Final sum is:", partialSum)
